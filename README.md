# Payment app
第三方線上支付付款流程

## 使用技術
- React
- React Router
- SCSS
- HTML5
- JavaScript

## 功能
- 可以選擇付款方式
- 選擇超商取貨付款可以選擇超商
- 付款資訊若填寫錯誤會跳出錯誤訊息
- 付款完成可以瀏覽訂單
- 支援 RWD
